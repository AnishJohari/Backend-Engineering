ls :> Lists all files and directories in the current directory.

cat file :> Displays the content of the specified file on the terminal.

grep txt file :> Searches for the text txt inside a file and shows matching lines.

cd projects :> Changes the current directory to projects.

pwd :> Displays the present working directory.

rm file.txt :> Deletes the file named file.txt.

cd folder :> Moves into the specified folder.

cd .. :> Moves back to the parent directory.

clear :> Clears the terminal screen.

cd ~ :> Moves to the user's home directory.

echo hello > a.txt :> Writes the text hello into the file a.txt.

mv a.txt b.txt :> Renames or moves a.txt to b.txt.

ls -i :> Lists files and directories along with their inode numbers.

rm b.txt :> Deletes the file b.txt.

mkdir -p a/b/c :> Creates nested directories a, b, and c in a single command.
